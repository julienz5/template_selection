Pouces (par défaut)

Deux pouces cliquables seront proposés aux destinataires directement dans l'email: un pouce vers le haut et un pouce vers le bas. Le pouce vers le haut redirigera vers le lien choisi lors de l'envoi de l'email. Le pouce vers le bas redirigera vers une page Web sur laquelle le destinataire pourra exprimer son avis.

Étoiles

Un boutton "Laisser un avis" sera proposé aux destinataires dans l'email. Un clic sur ce boutton redirigera vers une page Web de notation. Les destinataires pourront déposer une note de 1 à 5 étoiles. En fonction de cette notation, vous pouvez définir à partir de quelle note rediriger le destinataire vers le lien choisi lors de l'envoi de l'email ( par défaut, la redirection se fait à partir de 3 étoiles sur 5).

Smileys

Trois smiley cliquables seront proposés aux destinataires directement dans l'email: un smiley souriant, un smiley neutre et un smiley mécontent. Le smiley souriant redirigera vers le lien choisi lors de l'envoi de l'email. Le pouce mécontant redirigera vers une page Web sur laquelle le destinataire pourra exprimer son avis.
